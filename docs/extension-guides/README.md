# 插件指南
通过[Hello World]()学习过基本的VS Code插件API之后，现在是时间搞搞真正的插件了。在[插件功能]()章节，在更宽泛的层面上介绍了插件**能做**些什么，本章则细化了各个功能点，并提供了详细的代码和VS Code API的解释。

在每个指南-示例组合中，你将会看到：
- 贯穿整个代码的注释
- 使用示例插件的gif或者图片
- 运行示例插件指引
- 使用过的VS Code API列表
- 使用的配置点列表
- 真正的组合插件示例
- API概念解释

## 指南和例子

下面是一份指南和例子的表格，虽然每个指南都要有示例代码，但是一些示例目前暂时还没有与之匹配的指南。

每个例子都至少阐释了一个VS Code API或者发布内容配置的使用。
